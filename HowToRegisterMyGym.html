1

Email us

In the email, please include the name of your gym and the branch of the gym you are wanting to register. One of our team members will send you a code to use for the registration process.

2

Use your code when registering

Underneath these steps in the button that takes you to the gym register page. On this page, enter all these details and use the code you were given.

3

Upload a csv file of your database

We will provide an example of what structure the database must follow. Once it the database is in this structure, you can upload the csv file and now your gym is ready to use the application.
