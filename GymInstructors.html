As an instructor of a gym, you want to be able to let everyone know you are unable to teach one of your classes. You want to be able to edit the classes you teach. You also want to be able to see everything announced at the gym. This apllication will help you with this all.

  • Cancel a class - If you can't teach your class, let everyone who booked a spot know that it won't be happening anymore by cancelling it. You do this on the class details page.
  • Edit a class - If you're unhappy with any of the details of the classes you teach, you can edit them. This is also done through the class details page.
  • Be informed - You can also see all announcements made by the managers on the announcements page.
