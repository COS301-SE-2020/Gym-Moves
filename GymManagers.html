As a gym manager, you want to have that communication with all gym goers so that everyone knows of important events taking place at the gym. You want to be able to control the classes at the gym. You also want to know all the statistics of the gym so you know how to improve what is already there. This application helps you to do all that.

  • Send announcements - Keep everyone at the gym informed by sending them anouncements of what is happening at the gym. You can find the screen to send announcements as an option on the menu.
  • Add classes - You can add a new class for the gym members to enjoy. You can do this from the screen where you can view all the classes.
  • Edit classes - You can edit a class's details if you are unhappy with them. You can do this from the class detail screen.
  • Remove classes - You can remove a class if you are unhappy with it. You can do this from the class detail screen.
  • View the gym's dashboard - You can use the login button above to login. The dashboard will give you all sorts of information about the gym to allow you to make decisions on how to improve everything.
